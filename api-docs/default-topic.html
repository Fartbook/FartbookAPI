Fartbook API Documentation Help

Welcome

This is the documentation for the Fartbook API.

How to start

  • In your terminal, type

    cd {the directory where Fartbook is} node app.js

    After running this, you should get this output:

    Fartbook API is running on port 3000

    Now, you can go to your browser, navigate to localhost:3000 and you'll see Fartbook.
