API Endpoints

Because i cannot use api-doc (Writerside wants OpenAPI), i will use the code tag instead for the API endpoints.

/api/fartbook [GET] - no params - Returns all posts in JSON

/api/fartbook [POST] - username, postText - Makes a post with username and postText.

/api/fartbook/:id/comment [POST] - username, text - Makes a comment on post in endpoint

/api/fartbook/:id/search [GET] - no params - Searches for posts
